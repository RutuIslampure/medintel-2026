## 🚀 Quick Start (3 steps)

### Option A — One-click setup (Recommended)
```bash
bash setup.sh
```

### Option B — Manual setup
```bash
# 1. Create virtual environment
python3 -m venv venv
source venv/bin/activate        # Mac/Linux
# .\venv\Scripts\activate       # Windows

# 2. Install dependencies
pip install -r requirements.txt

# 3. Generate data & train model
python generate_data.py
python train_model.py

# 4. Launch the app
streamlit run app.py
```

Then open **http://localhost:8501** in your browser.

## ✨ Features

| Feature | Description |
|---|---|
| **🤖 AI Risk Prediction** | Gradient Boosting model classifies patient risk as Low/Moderate/High |
| **📖 Health Story Timeline** | Converts raw vitals into plain-English narrative for clinicians |
| **🔮 What-If Simulator** | Virtually test treatments (O₂ therapy, antihypertensives, etc.) |
| **🧠 Explainable AI** | Shows which vitals are driving the risk score |
| **🗺️ Risk Heatmap** | Hospital-wide ward-level risk visualization |
| **📊 Analytics** | Population trends, age-group & condition breakdown |
| **🚨 Alert System** | Real-time high-risk patient alerts on dashboard |

---

## 🧠 ML Model

- **Algorithm**: Gradient Boosting Classifier (scikit-learn)
- **Features**: Heart rate, BP, O₂ saturation, temperature, respiratory rate, glucose, WBC, creatinine, hemoglobin, age, gender
- **Output**: Risk label (Low / Moderate / High) + probability scores
- **Validation**: 5-fold cross-validation

---

## 🔮 What-If Scenarios Available

- **Oxygen Therapy** — Simulates supplemental O₂ administration
- **Antihypertensive** — Models BP medication effects
- **Fluid Resuscitation** — IV fluid bolus simulation
- **Fever Reduction** — Antipyretic therapy (Paracetamol)
- **Insulin Therapy** — Glucose management simulation
- **Custom** — Manually adjust any vital sign
